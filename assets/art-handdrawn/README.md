# Lil8TD Hand-Drawn Atlas V1

`lil8td-handdrawn-v1.png` is the first corrective hand-drawn pixel-art pass after the earlier atlas still read too much like a tidy prototype board.

Source/rebuild helper: `scripts/generate-handdrawn-art.py`.

Sprite families present:

- 4 hand-painted moss/buildable grass tiles with large brush fields, weeds, stones, and irregular outlines
- 4 hand-painted compacted-road/path tiles with meandering dirt centers, side stones, and stronger material contrast
- 3 distinct tower sprites:
  - arrow: crooked stone hut with oversized bow/bolt and red banner
  - burst: squat cannon/cauldron with ember diamond
  - frost: stone base with tall cyan crystal obelisk
- 3 distinct enemy sprites:
  - grunt: tan toy soldier with cap and shield
  - runner: red low forward-leaning runner/scarf shape
  - brute: wide purple horned ogre silhouette
- UI corner ornament and role icon pieces for frame/world consistency

The atlas keeps the same dimensions and sprite coordinates as the previous V1 atlas so GitHub Pages remains static and no heavy build step is required.
