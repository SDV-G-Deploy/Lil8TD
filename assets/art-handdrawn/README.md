# Lil8TD Hand-Drawn Atlases

`lil8td-sprite-set-v2.png` is the active Sprite Set V2 atlas. It replaces the first corrective V1 pass with calmer clustered terrain, a clearer handcrafted path language, and more heroic tower silhouettes.

`lil8td-handdrawn-v1.png` remains in the repo as a fallback/reference artifact.

Source/rebuild helper: `scripts/generate-handdrawn-art.py`.

Sprite Set V2 families present:

- 4 hand-painted moss/buildable grass tiles organized as family A base moss, family B warm dry moss, family C cool shaded moss, plus one quiet bridge variant
- 4 hand-painted compacted-road/path tiles with broader dirt centers, darker shoulders, sparse side stones, and worn highlight bands
- 3 hero tower sprites:
  - arrow: tall mast, oversized bow crescent, bolt tip, restrained crimson pennant
  - burst: broad kettle-cannon body, ember core, strongest horizontal mass
  - frost: tallest crystal crown/spire, teal/cyan identity, colder silhouette contrast
- 3 distinct enemy sprites retained from the V1 personality pass
- UI corner ornament and role icon pieces for frame/world consistency

The V2 atlas keeps the same dimensions and sprite coordinates as V1 so GitHub Pages remains static and no heavy build step is required.
